Codecraft连接Wio Terminal与Azure IoT

简介​

在这里，我们将展示如何将Codecraft与基于Wio Terminal的Azure IoT连接，并将传感器应用于物联网项目。 Codecraft 是由Seeed开发的可在线和离线使用的图形化可编程应用程序。 Azure IoT是由Microsoft开发的托管物联网应用平台，具有安全性，随着业务增长而扩展，并与现有的业务应用集成。

入门指南​

硬件​

所需硬件

在此演示中，您将需要以下设备：

  • WioTerminal x1
  • Type-C 数据线 x1
  • 计算机 x1

硬件连接

通过Type-C数据线将其连接到计算机。

软件​

所需软件

  • Codecraft
  • Azure IoT

教程​

在这里，我们将展示如何在Codecraft中使用Wio Terminal，并将Codecraft与Azure IoT连接，以训练机器学习模型并使用它。该项目基于Arduino平台，这意味着需要Arduino IDE和各种Arduino库。如果您是第一次使用Wio Terminal，请查看一个快速入门指南来 使用 Wio Terminal.

  • 步骤 1 连接Wio Terminal与PC。
  • 步骤 2 打开 Codecraft 网站 并与Wio Terminal连接。

选择 "Wio Terminal"

选择 "Connect" , 这可能需要安装CodecraftAssistant（如果这是您第一次使用Codecraft）。

正如您所看到的，这里需要设置Wi-Fi和 Azure IoT Central 范围 id Wi-Fi必须与您的环境相同，下一步我们将填写Azure IoT信息。

  • 步骤 4 打开 Azure 网站 并 creat a new app

点击左侧的 "Build" ，这里的信息是自定义的，并不是非常重要。

  • 步骤 5 点击 “Administration”-->"Device connection"-->"SAS-IoT-Device" 来添加Wio Terminal。
